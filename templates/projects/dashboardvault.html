Dashboard Vault

Dashboard Vault is a centralized web platform where I upload and share my Data Analysis Dashboards developed using Power BI and Tableau. It features a secure admin login for uploading content and offers viewers a seamless way to interact with visual data insights.

  • 📊 Showcases multiple dashboards with interactive visualizations.
  • 🔐 Admin Login to securely upload and manage dashboards.
  • 💡 Built using Flask (Python) with frontend animations for a smooth UI experience.
  • 🖥️ Optimized for desktop users; mobile support coming soon.
  • 📂 Ideal for recruiters or collaborators to explore live analytics in real-time.
