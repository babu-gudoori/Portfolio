Portfolio Website

Overview

This personal portfolio website is designed to present my technical skills, projects, and professional background in an interactive and engaging format. It features a sleek UI with responsive animations, and it integrates backend logic to store and manage data efficiently.

Features

  • Modern responsive design with animated sections and transitions
  • Interactive chatbot assistant named Riya (CSV-trained mini AI)
  • Admin login with password-protected dashboard

  • Triple-click on logo to reveal hidden admin login

Chatbot & Admin Panel

The chatbot named Riya is a lightweight assistant that can answer queries based on a preloaded CSV dataset. It supports text-based interaction.

The Hire Me form allows visitors to send project or job requests, which are stored securely in a MySQL database. To access this data, the admin must:

  • Click the logo three times to trigger the hidden admin login
  • Enter valid admin credentials (username & password)
  • Access the dashboard to view all form submissions
