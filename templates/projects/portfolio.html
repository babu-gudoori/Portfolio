Smart Personal Portfolio Website

This project is a dynamic and intelligent portfolio platform designed to highlight my capabilities as a developer. It combines AI-powered interaction with practical features like admin login, email integration, and data management via CSV.

  • ⚡ Fully responsive UI with smooth animations and modern layout.
  • 🤖 Integrated chatbot Riya, trained using CSV to respond intelligently to user queries.
  • 📝 “Hire Me” form that sends user submissions directly to the admin’s email and stores them in a CSV file for backup.
  • 🔐 Hidden Admin Dashboard – accessed by triple-clicking the logo – lets the owner view and manage hire requests.
  • 💬 Secure backend developed using Flask and Python, ensuring data handling and form processing.
